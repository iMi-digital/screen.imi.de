Freigegeben Bildschirm anschauen

Das Anschauen eines freigebenen Bildschirms funktioniert ohne Installation. Einfach den Code eingeben und "Freigebenen Bildschirm öffnen" klicken. Nachdem der Bildschirm angezeigt wird, können Sie die Anzeige über das Icon in der unteren rechten Ecke des Bildes maximieren.

Freigeben des eigenen Bildschirms oder eines Fensters

Das Freigeben Ihres Bildschirms oder einzelner Fenster funktioniert nur in Chrome oder Firefox nach Installation eines Plugins.
